Projects

Cryptocurrency Trading Bot

This year, I have been working on creating a crytpocurrency bot that uses the Binance Exchange with a team of students in order to profitably trade coins. The bot is built in Python and uses the Binance API in order to keep track of the market and analyze which coins can be traded for a gain. Currently, the bot uses SMA (small moving averages) and RSI (relative strength index) as tools and employs a short term, long term memory neural net that uses the past hour of data to predict the next 15 minutes. The code skeleton can be found on my Github.

Seam Carver

Priority Heap

A visual and functional representation of a priority heap built in Java that takes in integer keys and string values. This project can upheap, downheap, find the minimum element, replace any element in the heap and properly reorder the elements, keep track of size, etc.
